# PrepWise AI – Smart Study Planner

## What is This Project About?

This project helps students understand how to build a complete AI-powered full-stack application using Next.js and Supabase. It combines frontend development, backend logic, database storage, and AI integration into one practical project.

## Purpose

To build a smart study assistant that makes exam preparation more structured, personalized, and efficient.

## Learning Outcomes

- Learn routing, components, and API routes in Next.js
- Understand database operations using Supabase
- Integrate Groq API/other LLM for AI-powered study planning
- Store and retrieve user data efficiently
- Deploy applications using Vercel

## Skills Developed

- Full-stack web development
- API integration
- Database handling
- AI implementation
- Project deployment

## What You Have to Do (Step-by-Step)

**Project:** PrepWise AI – Smart Study Planner  
**Course:** Mastering Next.js – Build and Deploy Full-Stack Apps

PrepWise Lite is a simple AI-powered study planner where students enter a subject, topics, and exam date. The app generates a personalized study schedule and stores it for later use.

### Tech Stack

- Next.js
- Tailwind CSS
- Supabase
- Groq API / Any other LLM
- Vercel

### Day 1

1. Create a new Next.js project
2. Setup Tailwind CSS
3. Build a form: Subject, Topics, Exam Date

### Day 2

4. Create an API route
5. Connect Groq API / any other LLM API
6. Generate AI study plan

### Day 3

7. Connect Supabase
8. Save generated plans
9. Show saved plans
10. Deploy on Vercel
